Library Management Application
Welcome to the Library Management Application! This app allows you to manage both authors and books within a library system. You can add, edit, and delete information about authors and books, making it easy to keep your library organized.

Features
Author Management
Add Author: Input the author's name, birth date, and a brief biography.
Edit Author: Update the details of an existing author.
Delete Author: Remove an author from the list.
Book Management
Add Book: Input the book's title, author, ISBN, and publication date.
Edit Book: Update the details of an existing book.
Delete Book: Remove a book from the list.
Responsive Design
The application is designed to work well on both desktop and mobile devices.
How to Use the Application
Managing Authors
Adding an Author:

// Navigate to the Author Management section.

Fill in the form with the author's name, birth date, and biography.
Click the Add Author button to add the author to the list.
Editing an Author:

In the Author List, find the author you want to edit and click the Edit button.
The form will be populated with the author's current details.
Make any necessary changes and click Update Author to save the changes.
Deleting an Author:

In the Author List, find the author you want to delete and click the Delete button.
The author will be removed from the list.
